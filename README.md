# Informatik Repository

Dieses Repository dient der Verwaltung von Unterrichtsmaterialien und Hausaufgaben im Fach Informatik.

## Zweck
Hier werden Programmieraufgaben bearbeitet, gespeichert und abgegeben. Alle Schülerinnen und Schüler nutzen dieses Repository im Rahmen des Unterrichts.

## Struktur
- Jede Person besitzt einen eigenen Ordner.
- In diesem Ordner werden alle Aufgaben und Projekte abgelegt.

## Nutzung
- Arbeiten erfolgen ausschließlich im eigenen Ordner.
- Neue Aufgaben sollen klar benannt und strukturiert gespeichert werden.
- Änderungen müssen regelmäßig gespeichert werden.

## Versionskontrolle
- Nach jeder abgeschlossenen Änderung ist ein Commit durchzuführen.
- Commit-Nachrichten sollten kurz beschreiben, was geändert wurde.

## Hinweise
- Der Code sollte lesbar und nachvollziehbar sein.
